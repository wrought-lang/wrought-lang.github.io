Wrought

A convenient programming language representation for WebAssembly

Features

  • Compiles quickly and deterministically
  • Bridges the WASM explainability gap
  • Will enable decompilation of WASM modules

Examples

Wrought allows you to write easy to understand code that compiles directly to WebAssembly. This simple minimum function example compiles deterministically to WAT/WASM.

export fn min(left: u32, right: u32) -> u32 { if left < right { return left; } return right; }
